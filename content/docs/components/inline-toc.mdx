title: Inline Table of Contents
description: Give readers quick navigation within long documentation pages.
---

## Usage

Add the inline TOC to pages longer than 600 words. It automatically lists headings up to level three.

## Customization

Set sticky behavior, compact mode, and highlight colors to match your brand guidelines.

## Accessibility

Ensure the TOC appears before the main article in the DOM order so keyboard users can reach it quickly.
