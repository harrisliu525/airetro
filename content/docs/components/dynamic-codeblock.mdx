## Purpose

Display prompt recipes that teammates can copy into Retro Studio. The block renders preset name, prompt text, and optional tips.

## Editing content

Add code blocks in MDX using triple backticks. Use frontmatter variables to inject dynamic tokens such as current brand or campaign.

## Variations

Switch between light, dark, or terminal themes to match the surrounding content.
