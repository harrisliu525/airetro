---
title: Release Info Card
description: Summarize changelog entries and link to detailed release notes.
---

## Content structure

Display version number, release date, and short summary. Add a CTA button that links to the full changelog or blog post.

## Styling

Choose between minimal and spotlight layouts. Spotlight mode includes background gradients and iconography for big launches.

## Tips

Keep summaries under 120 characters and include at least one benefit statement for the user.
