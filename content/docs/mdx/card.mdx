## Structure

Each card includes an icon, title, description, and CTA link. Cards can be arranged in grids or carousels.

## Usage ideas

Promote new style packs, integrations, or success stories from the AI Retro community.

## Styling

Set elevation levels, border radius, and hover animations via theme tokens.
